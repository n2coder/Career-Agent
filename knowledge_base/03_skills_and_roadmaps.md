# Skills and Role Roadmaps for India IT Careers (Updated: February 12, 2026)

## Common Hiring Baseline Across Most Roles
- Programming: Python or JavaScript/TypeScript (role-dependent)
- SQL and data handling basics
- API design and debugging
- Git, Linux shell, and deployment fundamentals
- Communication: concise written updates and technical articulation

## A) Backend Engineer Roadmap (6-9 months)
1. Core foundations:
- Python (FastAPI/Django) or Node.js (Express/Nest)
- REST, auth, caching, queues, async fundamentals
2. Data layer:
- PostgreSQL schema/indexing/query tuning
- Redis basics and usage patterns
3. Reliability:
- Logging, metrics, retry/idempotency, error budgets
4. Ship projects:
- production-style API with auth + background workers
- one system-design case study with scale tradeoffs

## B) Data Engineer Roadmap (6-10 months)
1. Core data stack:
- SQL advanced, Python, ETL/ELT patterns
2. Platform tools:
- one warehouse (BigQuery/Snowflake/Redshift)
- orchestration fundamentals
3. Quality and governance:
- data validation, lineage basics, alerting
4. Ship projects:
- end-to-end batch and near-real-time pipeline demo

## C) AI/ML Engineer Roadmap (8-12 months)
1. Core stack:
- Python, NumPy/Pandas, model evaluation basics
2. Applied ML:
- feature engineering, model training, error analysis
3. GenAI and RAG:
- embeddings, retrieval quality, prompt strategy
4. MLOps basics:
- model serving, monitoring, drift and feedback loops
5. Ship projects:
- one domain model + one RAG app with measurable quality metrics

## D) Cloud/DevOps/SRE Roadmap (6-10 months)
1. Linux + networking fundamentals
2. Cloud core services (AWS/Azure/GCP)
3. Docker + CI/CD + IaC basics
4. Observability + incident workflow
5. Ship projects:
- automated deployment pipeline and operations runbook

## E) Cybersecurity Track (8-12 months)
1. Security foundations:
- network, IAM, secure coding, threat basics
2. Practical tooling:
- scanning, logging, hardening workflow
3. Cloud security baseline:
- secrets, policies, misconfiguration controls
4. Ship projects:
- security audit checklist + remediation walkthrough

## Role + Skill Pairing that Converts Better
- AI + backend
- data engineering + cloud
- cybersecurity + cloud
- SRE + automation

## Proof-of-Work Checklist
- 3 serious projects with architecture notes
- 1 benchmark/metrics write-up
- 1 debugging or incident postmortem
- clean README, setup steps, and demo media
